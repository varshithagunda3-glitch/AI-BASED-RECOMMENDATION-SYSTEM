*Task Description* :

This project implements a simple AI-based recommendation system using Java that suggests products to users based on their preferences. The system analyzes user ratings for different products and recommends items that the user has not rated but other users have liked.
The program uses sample user data, where each user has rated certain products such as Laptop, Phone, Tablet, and Headphones. When a user enters their name, the system examines the ratings given by other users and calculates an average score for products the selected user has not rated. Based on these scores, the system recommends the most suitable products.
This approach follows the basic concept of collaborative filtering, which is widely used in recommendation systems. The idea is that if users have similar preferences, products liked by one user may also be liked by another user.

*How the System Works* :

The program stores user-product ratings using Java data structures like HashMap.

Each user has rated different products with values from 1 to 5.

The system asks the user to enter a username (User1, User2, or User3).

It compares the selected user’s ratings with ratings from other users.

Products that the selected user has not rated are analyzed.

The program calculates the average rating score of those products from other users.

Finally, the system displays the recommended products with their scores.
